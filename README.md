# viriciti-test-workshop

This application was made with the purpose of showing testing examples with Vue + Jest + vue-test-utils. The purpose is to be shown during a presentation on ViriCiti.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your unit tests

```
npm run test:unit
```

### Lints and fixes files

```
npm run lint
```
